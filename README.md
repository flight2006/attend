#简介
本项目是一个基于Java的第三方微信公众号管理系统，可用于微型企业人事管理、考勤管理、部门通知留言、员工查看考勤记录等，项目前台为微信公众号，员工关注该公众号可以查询考勤记录、部门主管留言等信息；后台是一个后台管理页面，可以用于简单的人事管理、查询考勤记录、主管留言等。

#框架
本项目使用SpringMVC + Spring Framework + MyBatis框架，后台页面显示使用bootstrap，自适应移动端页面(虽然对于后台管理然并卵)。
